# Segunda Etapa

En esta **Segunda etapa** la aplicación se va a desarrollar en **React y TailwindCSS** del lado del frontend, **NestJS y PostgreSQL + TypeORM** en el backend

---

### Instalación

1. Clonar repositorio
2. Copiar `.env.examplo` → `.env`
3. Crear DB `falta-uno`
4. Correr comando `npm run start:dev`
